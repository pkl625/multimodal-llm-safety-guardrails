## L1 大类方向

### 输入安全

#### 概述
输入安全是多模态大模型安全的第一道防线，关注如何识别和阻止恶意输入在到达模型核心之前。

#### L2 子类与L3 前沿点

| L2 子类 | L3 前沿点 | 研究热度 | 成熟度 |
|---------|-----------|----------|--------|
| **提示注入防御** | 直接注入检测 | 🔥🔥🔥🔥 | 中 |
| | 间接注入检测 | 🔥🔥🔥🔥🔥 | 低 |
| | 跨模态注入 | 🔥🔥🔥 | 低 |
| **越狱攻击检测** | 文本越狱识别 | 🔥🔥🔥🔥 | 中 |
| | 视觉越狱识别 | 🔥🔥🔥🔥🔥 | 低 |
| | 多轮对话越狱 | 🔥🔥🔥🔥 | 低 |
| **多模态输入验证** | 图像-文本一致性 | 🔥🔥🔥 | 中 |
| | 音频-文本一致性 | 🔥🔥 | 低 |
| | 视频时序验证 | 🔥🔥 | 低 |

#### 关键挑战

1. **跨模态语义冲突**：当图像和文本传递矛盾信息时，模型如何正确识别意图
2. **隐式恶意意图**：有害内容被分散到多个模态，单模态检测失效
3. **上下文依赖**：相同输入在不同上下文中安全性不同

#### 代表性工作

- **CASA** (2026): 条件解码策略，利用内部表示预测安全token
- **Prompt Guard** (Meta): 基于mDeBERTa的注入检测，TPR>99%
- **Azure Prompt Shields**: 直接和间接注入检测

### 模型行为安全

#### 概述
模型行为安全关注模型内部的安全机制，如何在参数层面和推理层面确保模型行为符合安全规范。

#### L2 子类与L3 前沿点

| L2 子类 | L3 前沿点 | 研究热度 | 成熟度 |
|---------|-----------|----------|--------|
| **安全对齐** | RLHF安全微调 | 🔥🔥🔥🔥 | 高 |
| | 多模态偏好学习 | 🔥🔥🔥🔥 | 中 |
| | 安全约束优化 | 🔥🔥🔥 | 低 |
| **推理时干预** | 激活引导 | 🔥🔥🔥🔥 | 中 |
| | 零空间投影 | 🔥🔥🔥 | 低 |
| | 表示移位移除 | 🔥🔥🔥 | 低 |
| **内部表示分析** | 拒绝方向识别 | 🔥🔥🔥🔥 | 中 |
| | 安全神经元定位 | 🔥🔥🔥 | 低 |
| | 层间动态分析 | 🔥🔥 | 低 |

#### 关键挑战

1. **安全-效用权衡**：增强安全往往降低模型有用性
2. **对齐遗忘**：领域微调导致安全对齐遗忘
3. **稀疏安全路径**：安全行为集中在少量参数，易被攻击

#### 代表性工作

- **SafeNeuron** (2026): 神经元级安全对齐，冻结安全神经元
- **NullSteer** (2026): 零空间投影，安全与效用兼得
- **Safe RLHF-V** (2025): 多模态安全强化学习框架

### 输出安全

#### 概述
输出安全是最后一道防线，确保模型生成的内容符合安全标准，不包含有害、虚假或敏感信息。

#### L2 子类与L3 前沿点

| L2 子类 | L3 前沿点 | 研究热度 | 成熟度 |
|---------|-----------|----------|--------|
| **内容过滤** | 毒性检测 | 🔥🔥🔥🔥 | 高 |
| | 偏见检测 | 🔥🔥🔥 | 中 |
| | PII检测 | 🔥🔥🔥🔥 | 中 |
| **幻觉检测** | 事实一致性验证 | 🔥🔥🔥🔥 | 中 |
| | 来源可追溯性 | 🔥🔥🔥 | 低 |
| | 不确定性量化 | 🔥🔥🔥 | 低 |
| **输出后处理** | 安全重写 | 🔥🔥🔥 | 中 |
| | 敏感信息脱敏 | 🔥🔥🔥🔥 | 中 |
| | 结构化输出验证 | 🔥🔥🔥 | 中 |

#### 关键挑战

1. **幻觉与安全的界限**：模型拒绝回答 vs 模型编造回答
2. **部分合规**：模型部分回答有害内容
3. **多语言输出安全**：低资源语言的安全检测

#### 代表性工作

- **Llama Guard 3**: 模块化输入-输出分类器，支持13+风险类别
- **OpenAI Moderation API**: omni-moderation-latest支持文本+图像
- **LLM Guard**: 20个输出扫描器，覆盖多种风险

### 系统级安全

#### 概述
系统级安全关注多模态大模型在实际部署环境中的安全，包括代理系统、多轮对话、工具使用等复杂场景。

#### L2 子类与L3 前沿点

| L2 子类 | L3 前沿点 | 研究热度 | 成熟度 |
|---------|-----------|----------|--------|
| **护栏编排** | 多护栏协同 | 🔥🔥🔥🔥 | 中 |
| | 可编程护栏 | 🔥🔥🔥🔥 | 中 |
| | 延迟优化 | 🔥🔥🔥 | 中 |
| **多代理安全** | 代理间通信安全 | 🔥🔥🔥 | 低 |
| | 工具使用安全 | 🔥🔥🔥🔥 | 中 |
| | 记忆治理 | 🔥🔥🔥 | 低 |
| **供应链安全** | 模型来源验证 | 🔥🔥 | 低 |
| | 数据投毒防御 | 🔥🔥🔥 | 中 |
| | 后门攻击检测 | 🔥🔥🔥 | 低 |

#### 关键挑战

1. **多轮对话安全**：恶意意图在对话中逐步暴露
2. **工具滥用**：模型生成有害工具调用
3. **代理权限边界**：防止权限升级和越权操作

#### 代表性工作

- **NeMo Guardrails**: NVIDIA的护栏编排框架
- **SSGM Framework** (2026): 记忆治理框架
- **LLaVAShield** (2025): 多轮多模态对话安全

## 热点研究主题

### 主题1: 多模态越狱攻击 🔥🔥🔥🔥🔥

**研究热度**: 最高

**核心问题**: 如何利用多模态特性绕过安全机制

**主要方向**:
- 视觉叙事攻击（漫画、故事板）
- 分散式语义攻击（多图像分散）
- 跨模态线索纠缠
- 游戏化场景攻击

**关键论文**:
- TreeTeaming (2026): 层次化策略探索
- MIDAS (2026): 多图像分散与重构
- GAMBIT (2026): 游戏化越狱

**未来方向**:
- 音频-视觉联合攻击
- 视频时序攻击
- 3D场景攻击

---

**核心问题**: 如何自动化发现和验证安全漏洞

**主要方向**:

- 多代理协作红队
- 策略树动态扩展

**未来方向**:
- 全自动漏洞挖掘
- 攻击策略知识库
- 红队-蓝队对抗训练

## 开放研究问题

### 高优先级问题

1. **跨模态安全一致性**
   - 问题：为什么文本对齐的安全机制在视觉模态上失效？
   - 挑战：模态间安全知识迁移
   - 潜在方向：模态无关安全表示学习

2. **安全-效用帕累托前沿**
   - 问题：如何量化并优化安全与效用的权衡？
   - 挑战：缺乏统一评估标准
   - 潜在方向：多目标优化框架

3. **自适应攻击防御**
   - 问题：如何防御不断进化的攻击策略？
   - 挑战：攻击面持续扩大
   - 潜在方向：在线学习+威胁情报

### 中优先级问题

4. **多轮对话安全**
   - 上下文安全累积
   - 意图渐进暴露检测
   - 对话状态安全监控

5. **低资源语言安全**
   - 多语言安全对齐
   - 文化差异与安全标准
   - 翻译引入的安全风险

6. **实时安全保证**
   - 推理时安全验证
   - 延迟-安全权衡
   - 流式内容安全

### 新兴问题

7. **具身智能安全**
   - 视觉-语言-行动模型安全
   - 物理世界交互风险
   - 机器人安全护栏

8. **多代理系统安全**
   - 代理间协作安全
   - 权限边界维护
   - 集体智能风险
